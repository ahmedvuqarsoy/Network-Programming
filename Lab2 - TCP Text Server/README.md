## Usage

**Options** for the program:

```
-h, --help			Help Menu
--host				Destination IP Address
-m					Client working mode
-p 					Destination Port
-f 					File
-a 					Additional file
```


**Server** mode:

```
python3 TextService.py server -p <PORT>
```

**Client** mode:

```
python3 TextService.py client -h <DST-IP> -p <PORT> -m <MODE> -f <FILE> -a <ADDITIONAL>
```
